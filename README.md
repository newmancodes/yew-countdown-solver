# Countdown Solver

[![CI/CD](https://github.com/newmancodes/yew-countdown-solver/actions/workflows/main.yml/badge.svg)](https://github.com/newmancodes/yew-countdown-solver/actions/workflows/main.yml)

A web application built with [Yew](https://yew.rs/) that solves the numbers round of the popular game show "Countdown". It allows users to input a set of numbers and a target number, and then computes possible solutions using the basic arithmetic operations (addition, subtraction, multiplication, and division).
